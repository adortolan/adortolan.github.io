Projetos Profissional e Acadêmicos

Aqui estão alguns dos projetos em que trabalhei recentemente atuando profissionalmente e como forma de estudo. Cada um deles representa uma oportunidade de aprendizado e crescimento profissional.

  • Softplan - SAJ Procuradorias atuando como desenvolvedor Delphi

    O SAJ Procuradorias da Softplan é um sistema de automação de processos judiciais e administrativos consultivos voltado especificamente para as Procuradorias. Ele visa automatizar, simplificar e acelerar a rotina de trabalho, oferecendo funcionalidades para a gestão de Execução Fiscal, Contencioso Judicial e processos administrativos consultivos. Durante dois anos e meio, trabalhei como desenvolvedor Delphi na sustentação deste sistema, atuando diretamente na manutenção e evolução da aplicação, utilizando bancos de dados Oracle e SQL Server para garantir a integridade e a performance dos dados. O sistema busca integrar as Procuradorias com os Tribunais, otimizar prazos, compartilhar conhecimento entre procuradores e aumentar a eficiência e a transparência na atuação jurídica do setor público.

  • Softplan - Gestão de Cobrança atuando como desenvolvedor React e Spring Boot.

    Como desenvolvedor, durante um ano e meio, trabalhei ativamente no sistema de gestão de cobrança da Softplan, onde minhas responsabilidades se concentraram no desenvolvimento utilizando as tecnologias React para o frontend e Spring Boot para o backend. No frontend, fui responsável por construir interfaces de usuário interativas e responsivas, utilizando o React para criar componentes reutilizáveis e gerenciar o estado da aplicação. No backend, desenvolvi APIs robustas e escaláveis com Spring Boot, garantindo a comunicação eficiente com o frontend e a lógica de negócios da aplicação de gestão de cobrança. Essa experiência me proporcionou um conhecimento prático e aprofundado no desenvolvimento full-stack com essas tecnologias, contribuindo para a criação e manutenção de um sistema crítico para a Softplan.

  • DSCommerce é um sistema de venda de produtos

    O DSCommerce é um sistema de venda de produtos desenvolvido com a robusta e eficiente tecnologia Spring Boot. Criado pela DevSuperior como uma plataforma de aprendizado prático e abrangente, tive a oportunidade de mergulhar em seu desenvolvimento como parte da minha jornada de aprimoramento profissional. Através deste projeto, pude consolidar meus conhecimentos em construção de backends escaláveis e performáticos com Spring Boot, explorando desde a modelagem de entidades e a persistência de dados com Spring Data JPA, até a criação de APIs RESTful bem definidas e seguras. O DSCommerce representou um valioso laboratório para aplicar conceitos avançados do ecossistema Spring, reforçando minha capacidade de desenvolver soluções de e-commerce completas e aderentes às melhores práticas do mercado.

  • Miniblog - Plataforma de publicação de blogs simples e intuitiva.

    O Miniblog é uma aplicação web desenvolvida para facilitar a publicação e o gerenciamento de posts em formato de blog. Com uma interface amigável e intuitiva, permite que usuários criem, editem e excluam publicações de maneira prática. O frontend foi construído utilizando desenvolvido com foco em autenticação, CRUD de postagens e integração com Firebase, utilizando o ecossistema moderno do Next.js. O Miniblog é ideal para quem deseja compartilhar ideias, artigos ou experiências de forma rápida e organizada, integrando boas práticas de desenvolvimento full-stack.
